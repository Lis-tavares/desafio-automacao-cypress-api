# Desafio de Automação API - Trello

## Objetivo

Validar o retorno da API Trello conforme especificado no desafio técnico.

## Tecnologias Utilizadas

- JavaScript
- Cypress
- Node.js

## Cenário Automatizado

- Realizar requisição GET para a API Trello.
- Validar status code 200.
- Exibir e validar o conteúdo do campo `name` da estrutura `list`.
